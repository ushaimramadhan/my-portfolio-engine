A robust fullstack web application designed to transform a static personal website into a dynamic Content Management System (CMS). This project allows developers to manage their professional portfolio through a secure admin dashboard.

## Tech Stack
- **Frontend**: React.js + Vite
- **Backend**: Node.js + Express
- **Database**: PostgreSQL
- **Containerization**: Docker & Docker Compose
- **API**: RESTful API

## Key Features (MVP)
### Public View
- **Home/Hero**: Professional introduction.
- **Project Showcase**: Dynamic project cards fetched from the database.
- **Contact Form**: Direct messaging system saved to the database.

### Admin Dashboard (Protected)
- **Authentication**: Secure login for the owner.
- **Project Manager**: Full CRUD (Create, Read, Update, Delete) operations for projects.
- **Inbox**: Message management system to track inquiries.

## Project Structure
This repository uses a monorepo structure:
- `/server`: Backend kingdom (Node.js + Express).
- `/client`: Frontend kingdom (React + Vite).
- `docker-compose.yml`: Database & pgAdmin orchestration.
